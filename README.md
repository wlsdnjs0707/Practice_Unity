## # 주요 기능

### **1. 플레이어 세팅**
  - 플레이어 이동 (WASD로 이동, Shift로 달리기)
  - 공격 (마우스 좌클릭)
  - 점프 (스페이스바)
  - 3인칭 카메라 (Left Alt로 둘러보기)

### **2. 이벤트**
  - 공격 이벤트: Collider 충돌시 Death 애니메이션 출력
